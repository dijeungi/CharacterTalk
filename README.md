# 📝 캐릭톡

**CharacterTalk**은 AI 캐릭터와 자유롭게 대화할 수 있는 **챗봇 플랫폼**입니다.  
사용자는 자신만의 캐릭터를 생성하고, 원하는 주제로 대화방을 만들어 상호작용할 수 있습니다.

## 🎯 주요 기능

- 🎭 AI 캐릭터 생성 및 커스터마이징
- 💬 캐릭터와의 실시간 대화 기능
- 🗂️ 다중 대화방 생성 및 관리
- 🔐 사용자별 캐릭터/대화방 접근 제어

## 🛠️ 기술 스택

- Next.js 15 (App Router)
- TypeScript
- Python (Django)
- Module CSS

## 🚀 실행 방법

```bash
npm install
npm run dev
```
